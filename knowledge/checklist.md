# Trading Checklists

## Pre-Session Checklist (15 minutes before session)

### Market Preparation
- [ ] Check today's economic calendar — note all events with times in CET
- [ ] Identify any high-impact events (CPI, NFP, FOMC, GDP, PMI) — mark no-trade zones
- [ ] Note prior day levels: PDH = ___, PDL = ___, PDC = ___
- [ ] Note overnight levels: ONH = ___, ONL = ___
- [ ] Note round number levels nearby: ___
- [ ] Check where price is relative to PDC (above = bullish lean, below = bearish lean)
- [ ] Check overnight range width: narrow (< 10 pts) / normal (10-20 pts) / wide (> 20 pts)
- [ ] Determine daily bias: Long / Short / Neutral — Reason: ___

### Personal Preparation
- [ ] How do I feel? Focused / Tired / Emotional / Distracted
- [ ] If not focused: should I trade today? (be honest)
- [ ] Review yesterday's journal notes — any carryover lessons?
- [ ] Set daily loss limit alert: $55
- [ ] Remind myself: max 4 trades, max 1 contract, follow the plan

### Platform Check
- [ ] Charts loaded with correct timeframes (5-min, 15-min)
- [ ] Key levels marked on chart
- [ ] VWAP indicator active
- [ ] Order entry ready (correct symbol, correct contract month)
- [ ] Stop entry mechanism ready (bracket order or manual)

## Trade Entry Checklist (Before EVERY trade)

### Setup Validation
- [ ] Which strategy is this? ORB / VWAP Reversion / Trend Pullback / Level Reaction
- [ ] Does this setup match my playbook EXACTLY? (Not "kind of" — exactly)
- [ ] Is this a valid trading session? (London AM or NY PM, not lunch/overnight)
- [ ] Is there a news event in the next 15 minutes? If yes → DON'T TRADE

### Risk Check
- [ ] Stop price defined: ___
- [ ] Stop distance: ___ points (___ ticks)
- [ ] Dollar risk: $___  (must be ≤ $25)
- [ ] Target price: ___
- [ ] Reward:Risk ratio: ___:1 (must be ≥ 1.5:1)
- [ ] Current daily P&L: $___ (must not be near -$55 limit)
- [ ] Trades taken today: ___ (must be < 4)
- [ ] Consecutive losses: ___ (if 2+, did I take a 30-min break?)

### Execution
- [ ] Direction: Long / Short
- [ ] Entry type: Limit / Stop / Market
- [ ] Entry price: ___
- [ ] Number of contracts: 1
- [ ] Stop order placed: Yes / No (MUST be Yes before entry)

### Final Gate
- [ ] Am I taking this trade because I see a VALID setup? (Not boredom, revenge, or FOMO)
- [ ] Would I take this trade if I had 10 contracts instead of 1? (Confidence check)
- [ ] If BOTH answers are YES → EXECUTE

## Trade Exit Checklist

### During the Trade
- [ ] Is price approaching my target? → Prepare to exit
- [ ] Has the trade been open > 30 min with no progress? → Tighten stop
- [ ] Has price moved 1.5× my risk? → Consider moving to breakeven
- [ ] Has the reason for my trade changed? → EXIT regardless of P&L
- [ ] Am I approaching session end? (21:30 CET for NY, 15:15 for London-only) → Look to exit

### After Exit
- [ ] Record the trade immediately:
  - Entry: ___  Exit: ___  Direction: ___
  - Stop: ___  Target: ___
  - P&L: $___ (+/-)
  - Strategy: ___
  - Held for: ___ minutes
  - Grade: A (perfect execution) / B (good, minor issues) / C (rule violation)
  - Notes: ___

## Emergency Checklist

### If You Hit Daily Loss Limit ($55)
1. Close all positions immediately
2. Close the trading platform
3. Do NOT re-open it today
4. Write in your journal what happened
5. Go do something else (exercise, walk, anything non-screen)
6. Review in the evening with a clear head

### If You Feel the Urge to Revenge Trade
1. Take your hand off the mouse
2. Close the order entry window
3. Stand up and walk away for 5 minutes
4. Ask: "Will this trade follow my plan?"
5. If the answer isn't an immediate, calm "yes" → DON'T TRADE

### If You Have a Massive Win
1. Log the trade
2. Consider stopping for the day (lock in the win)
3. Do NOT increase position size on the next trade
4. Stay humble — one good trade doesn't change your risk rules
